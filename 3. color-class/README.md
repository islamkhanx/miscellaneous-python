## Задание #5: уменьшение контраста

### Формулы для изменения контраста

Желаемая доля контраста
> $c = [0,1]$

Уровень контраста
> $cl = -256 \cdot  (1 - c)$

Коэффициент коррекции контраста

> $F = {259 \cdot (255 + cl) \over 259 \cdot(255 - cl)} $

Уровень цвета
> $L = [0,255]$

> Color($L_R$, $L_G$, $L_B$)

Уровень цвета с измененной контрастностью
> $L' = F \cdot (L - 128) + 128$
